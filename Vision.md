# VFW Post 8680 Event Manager

## Mission

Build the simplest, easiest-to-use event management application possible.

The application should require very little training.

Every feature should save time, reduce mistakes, and make running VFW events easier.

The software should feel more like an iPad app than a spreadsheet.

## Our Guiding Principles

• Touch first.
• Very little typing.
• Large buttons.
• Easy to read.
• Fast.
• Reliable.
• Offline first.
• Automatic backups.
• Designed by people who actually run events.

When in doubt...

Choose the simpler solution.
